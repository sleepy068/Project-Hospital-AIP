Installing:
1. Extract the zip file (to where ever if need be)
2. Copy the file 'Mod_Additional_Insurance_Companies' to your Project Hospital installation addon's folder (see below).

Steam Path: C:\Program Files (x86)\Steam\steamapps\common\Project Hospital\ProjectHospital_Data\StreamingAssets\Addons\*put mod here*

3. Done, launch the game and enjoy.

Troubleshooting:
- Mod didn't load?
Make sure the mod's path file is as follows:
C:\Program Files (x86)\Steam\steamapps\common\Project Hospital\ProjectHospital_Data\StreamingAssets\Addons\Mod_Additional_Insurance_Companies
In the mod file should be Database & Preview.png

- Bugs or issues?
